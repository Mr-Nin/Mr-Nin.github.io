What are command block creations?

Command block creations add cool custom features into your Minecraft world.

How to set up

  1. Copy the text for a command creation
  2. In Minecraft, run /give @s command_block
  3. Place the command block down and paste your command into it
  4. Once you power it, you will get a shulker box filled with a bunch of command blocks
  5. Place the command blocks in the numbered order. Make sure the arrows on the side of the command blocks align
  6. Power the repeating command block with a redstone block
  7. If there are any command blocks named "One Time Activation" just power those once
